Rationale

We will use React for our frontend implementation as:

React allows the most convenience for frontend development as three of the five members have previous built projects on this library.

React has the most amount of support and updated documentation for any reference purposes.

React is the fastest frontend library to render components into the main page. The features like React router,route and switch link make it easy for connecting various and features of the application.

What we will use React for:

  • To render user input and send it to the server.
  • For user to have a smooth and easy user interface using our application.
  • For user to navigate through the various features of our application.
